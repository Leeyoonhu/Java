
안녕하세요 전지적 군인 시점 게시판을 만든 이윤후, 정성근 입니다.

저희가 이 주제를 선택하게 된 이유는 군대에 커뮤니티 사이트들이 없어지면서 서로 소통할 수 있는 기회의 장을 만들고자 하였습니다.
요즘 군대에서 휴대폰이 사용이 가능함에 따라 사용자들의 증가를 예측하여 이 주제를 선택하게 되었습니다.

------
다음으로 역할 및 기능소개를 하겠습니다.
저희 조는 이윤후님이 Spring, Mysql, java script를 사용한 백엔드를 구현하였고, 정성근님은 HTML, css, bootstrap을 사용한
프론트앤드를 구현하였습니다. 또한
개발환경에서 윈도우 10을 사용하였고, 사용한 DB로는 My SQL을 사용하였습니다. 그리고 개발 지원도구로는
이클립스, vs코드, 깃허브를 사용하여 서로 소통하였습니다. 
-----
다음으로 메인페이지 입니다. 메인페이지의 구성은 상단쪽 헤더, 왼쪽 어사이드, 하단부에 푸터로 나누었고, 
가운데 메인에는 정보를 활용할 수 있는 유튜브 영상과 인기게시글, 최근게시글을 모아두어 한눈에 보기
용이하게 제작하였습니다. 그리고 하단부에는 사진게시판을 활용하여 반가운 얼굴들을 볼 수 있고, 
랭킹 보드를 통한 이용자들의 소소한 성취감도 만나볼 수 있도록 재미난 컨텐츠를 제작하였습니다.

-----
다음으로는 회원가입 페이지 입니다. 회원가입 페이지에서는 보이시는대로 한 가지라도 조건이 부합하지 않은 경우 
회원 가입이 진행되지 않으며, 아이디와 비밀번호, 전화번호의 경우 정규식 표현을 사용하여 제한을 두었고, 
아이디와, 닉네임, 전화번호는 ajax 비동기 통신을 활용하여 유효성 검사를 구현 하였습니다.

----
다음은 로그인페이지 입니다. 로그인 페이지에서는 ajax를 사용하여 사용자가 입력한 아이디와 패스워드의 값을
데이터베이스에서 조회하여 둘 다 일치할 경우에만 로그인이 가능하도록 설계하였습니다.
로그인이 성공했을 경우에는 사용자 정보를 HTTP세션에 저장하여 현재 사용자의 정보를 파악할 수 있도록 제작하였습니다.

----
만약에 아이디를 까먹었으면 어떻게 해야할까요? 그래서 준비한 아이디 찾기 입니다. 
아이디 찾기 페이지에서도 보시는바와 같이 유효성 검사를 진행하고 있으며 정보를 정확하게 입력하여
이용자들로부터 놓치고 있는 부분이 없는지 확인할 수 있도록 돕고 있습니다. 
정확한 값을 입력하였을 경우 상단에 알림을 통해 아이디를 알려주도록 서비스를 제공하고 있습니다.

----
비밀번호 찾기 입니다.
비밀번호 찾기는 아이디 찾기와 마찬가지로 유효성 검사를 진행하고 있으며, 비밀번호 찾기 질문의 경우에는
데이터베이스에 저장된 아이디를 입력하였을 경우 회원가입 당시에 설정되어있던 비밀번호 찾기 질문을 보여주어
이용자들이 이용하기 편리하게 답변만 넣을 수 있도록 구성 하였습니다. 

-----
이게 뭔가 싶죠? 저희가 좌측면에 회원 정보를 확인할 수 있도록 정보 제공창을 분류하여 만들었습니다.
비 로그인시 입니다. -- 회원가입시 군인이 아닐경우 -- 군인으로 가입할 경우인데, 군인일 경우에는 일기장과 전역일계산
버튼이 활성화되어 다른 회원과 차별화를 두었습니다.

----
지금부터는 게시판 관련 페이지 입니다.
첫번째로 보여지는 것은 자유게시판 입니다. 자유게시판은 비 로그인과 로그인 상태에 차이를 두어
상단에서 보시는바와 같이 비로그인 시에는 글쓰기 버튼이 보이지 않게 설계되었고
------
로그인을 했을 경우 글쓰기 버튼이 나와 자유롭게 글쓰기가 가능해 지도록 제작 하였습니다.
사진 게시판을 제외한 각 게시판들에서는 한 페이지당 15개의 게시글을 보여주고 있으며, 10페이지가 넘어갈 경우
next 버튼이 활성화 되어 다음페이지로 이동할 수 있도록 하였고, 다음페이지로 넘어갈 경우 previous 버튼이 활성화 되어
이전페이지로 돌아갈 수 있도록 설계 하였습니다. 
-----
질문 게시판과 정보게시판도 앞선 자유게시판과 마찬가지로 비로그인과 로그인의 차이를 두었고, 페이징처리를 통하여
사용자 입장에서 편리하게 정보를 얻어갈 수 있도록 구성 하였습니다.

-----
사진 게시판에서는 다른 게시판들과 다르게 한 페이지당 6개씩 사진을 배치하였고, 데이터 베이스의 저장된 이미지를
미리 볼 수 있게 하여 반가운 얼굴들을 보다 빨리 만나볼 수 있도록 구성 하였습니다.
그리고 아이콘을 활용한 조회수와 추천수를 확인해보며 인싸를 누려볼 수 있는 기회를 제공하고 있습니다.

다음은 전체게시판 입니다. 전체 게시판에서는 다른 게시판과 달리 모든 글을 한눈에 볼 수 있도록 설계하였고, 
그에 따른 검색기능이 가능하도록 구성하였습니다. 
제가 제목에 군대를 한번 검색해보겠습니다.

군대를 포함한 문장을 조회하여 보여주도록 하였습니다.

설계하였습니다. 
출석체크는 로그인을 하여야 출석이 인정되며 보시는바와 같이 비로그인일 경우나, 이미 출석한 경우에는 중복 출석이
불가능하도록 설정 하였습니다.

랭킹게시판에서는 데이터베이스에 등록된 유저의 경험치가 높은 순서대로 보여주도록 만들었으며,
그에 따른 랭킹 순위도 함께 볼 수 있도록 하였습니다. 또한 현재 로그인 한 사용자에 랭킹을 파악하기 쉽도록 배경색깔을
달리하여 보다 쉽게 확인할 수 있습니다.
